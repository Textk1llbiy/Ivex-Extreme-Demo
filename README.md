# Ivex-Extreme-Demo
This will import the Common Kali Linux base command to Windows. it does not yet still has a installer this is an EXTREME DEMO

Ivex stands for: Intuitive Virtual Enhancement eXtension

---- Usage ----

Until i create the realeas you can play with the source code. The realease shoud be out soon but for the one's wich want to know:

You need to have a cmq with or without the administrator previlages and the general command is:

	Ivex -[Command] -[Subcommand]

The commands are the common Linux command that i have implemented. to know more about the command i have implemented just use the command:

	Ivex -h
